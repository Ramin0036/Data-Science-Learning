# 🧠 Deep Learning

This section covers the fundamental concepts and practical applications of **Deep Learning**, from basic neural networks to advanced architectures for Computer Vision and Natural Language Processing.

Deep Learning enables machines to automatically learn complex patterns from large-scale data using multi-layer neural networks. Throughout this section, you will build deep learning models with modern frameworks and apply them to real-world tasks.

---

## 📚 Topics

### 📌 Neural Networks
Learn the fundamentals of **Artificial Neural Networks (ANNs)**, including perceptrons, activation functions, forward propagation, backpropagation, gradient descent, loss functions, and optimization algorithms.

### 📌 Convolutional Neural Networks
Explore **Convolutional Neural Networks (CNNs)** for image classification and computer vision applications. Topics include convolution, pooling, feature extraction, transfer learning, and popular CNN architectures.

### 📌 Multi-Channel Neural Networks
Study architectures that process **multiple input channels** simultaneously. Learn how to combine heterogeneous data such as images, tabular features, and metadata into a unified deep learning model.

### 📌 Text Processing with Bag of Words
Learn the fundamentals of **Natural Language Processing (NLP)** using the Bag of Words approach. This section covers text preprocessing, tokenization, vectorization, document-term matrices, and text classification.

### 📌 Text Processing with Word Embeddings
Discover dense vector representations of words using **Word Embeddings**. Learn techniques such as Word2Vec, GloVe, FastText, and embedding layers for building more powerful NLP models.

### 📌 Recurrent Neural Networks
Understand sequence modeling using **Recurrent Neural Networks (RNNs)** and their advanced variants including **LSTM** and **GRU**. Learn sequence prediction, time-series forecasting, and NLP applications.

### 📌 U-Net
Explore the **U-Net** architecture for semantic image segmentation. Learn encoder-decoder networks, skip connections, and applications in medical imaging and computer vision.
