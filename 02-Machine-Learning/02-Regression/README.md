# Linear and Non-Linear Regression Project

- Regression is one of the core topics in supervised machine learning. Its goal is to predict a continuous numerical value based on input data.
- In simple terms, regression tries to find a relationship between input variables (features) and an output value (target).

## Description

This repository contains implementations of various machine learning regression techniques. It covers both linear and non-linear models, providing a comprehensive guide and comparison for regression analysis.

### Features 
- **Linear Regression**: Simple Linear Regression.(uses only one input feature for prediction, such as predicting house price based only on area)
- **Linear Regression**: Multiple Linear Regression.(uses multiple input features, such as predicting house price based on area, number of rooms, construction year, and location)
- **Non-Linear Regression**: Polynomial Regression.(fits a curve to the data by adding powers of the input features)
- **Advanced Models**: Decision Tree Regressor, KNN Regressor and SVR.
- **Data Preprocessing**: Handling missing values, feature scaling, and data visualization.

### Tech Stack 
- Python 3.10.0
- Scikit-learn
- Pandas & NumPy
- Matplotlib & Seaborn
